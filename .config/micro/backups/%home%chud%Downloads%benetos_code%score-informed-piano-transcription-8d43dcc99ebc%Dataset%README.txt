Score-informed Piano Transcription Dataset - Modified Ground Truth

This set of MIDI files is an ammentment to the 'Score-informed Piano Transcription
Dataset' as published by Emmanouil Benetos in 2012. It contains modified/corrected
versions of the original ground truth MIDI files.

These modified files were first used in:

Sebastian Ewert, Siying Wang, Meinard Müller and Mark Sandler, "Score-Informed
Identification of Missing and Extra Notes in Piano Recordings", Proceedings of the
International Society for Music Information Retrieval Conference (ISMIR), New York,
USA, pp. 30-36, 2016.
